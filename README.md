# AI Chatbot

## Project Overview

This project is an AI Chatbot developed as part of the WeIntern AIML Week 2 Internship Assignment.

The chatbot can handle predefined user queries using an intent-response framework and provide relevant responses through a Streamlit-based web interface.

## Features

* 10 predefined intents
* Natural language text input
* Multi-turn conversation support
* Session-based chat history
* Fallback response for unknown queries
* Simple and user-friendly interface

## Technologies Used

* Python
* Streamlit
* JSON

## How to Run

1. Install dependencies:

pip install streamlit

2. Run the application:

python -m streamlit run app.py

3. Open the browser link generated by Streamlit.
